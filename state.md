# Glide Identity - Verifiable Credentials Website

## Project Summary
A landing page for Glide Identity's Verifiable Credentials platform, inspired by MATTR's design patterns with Glide branding.

## Live URLs
- **GitHub Pages**: https://eranhaggiag.github.io/glide-website-ideas/
- **Repository**: https://github.com/eranhaggiag/glide-website-ideas

## Sections (in order)
1. **Hero** - "Secure identity that flows seamlessly" with floating credential cards
2. **Stats** - 99.99% Verification Accuracy | 99.9% Uptime SLA | SOC 2 Type II
3. **How It Works** - Issue → Store → Verify flow
4. **Features** - 6 cards (Selective Disclosure, Cryptographic Security, Global Interoperability, Real-time Verification, Revocation Support, Developer APIs)
5. **Security** - Shield animation with security checklist
6. **SIM-Based Device Binding / MagicalAuth** - Carrier-grade cryptography, CAMARA API integration, carrier logos (T-Mobile, Verizon, AT&T, Vodafone, Telefónica, Orange)
7. **eIDAS 2.0 / European Digital Identity** - EUDI Wallet compliance, EU stars visualization, timeline (May 2024 → 2025 → Dec 2026 → 2030)
8. **Agentic Execution / Confidential Computing** - TEE diagram showing User Identity protected from AI Agent, Intel SGX & AMD SEV-SNP badges
9. **Use Cases** - Government, Finance, Education, Healthcare
10. **Standards** - W3C VC, SD-JWT, OID4VCI, OID4VP, DIF (with icons)
11. **CTA** - "Ready to build trusted experiences?"
12. **Footer** - Full navigation, social links

## Key Features
- Official Glide Identity logo (fingerprint with green center)
- Flowing background orbs with animations
- Wave dividers between sections
- SVG icons throughout (no emojis)
- Responsive design for mobile
- Smooth scroll reveal animations

## Technologies Referenced
- MagicalAuth (SIM-based cryptography)
- CAMARA API (carrier integration)
- eIDAS 2.0 / EUDI Wallet
- Trusted Execution Environments (Intel SGX, AMD SEV-SNP)
- W3C Verifiable Credentials, SD-JWT, OID4VCI, OID4VP
